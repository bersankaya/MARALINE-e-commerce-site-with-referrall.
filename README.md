
MARALINE – REFERRAL BASED E-COMMERCE PLATFORM

---

PROJECT OVERVIEW

* Maraline is an e-commerce platform developed with ASP.NET Core MVC and an integrated referral system.
* The platform supports product sales, order management, and user-based earnings tracking.
* A structured multi-level referral bonus system is implemented with strict business rules.
* The admin never earns any profit from the system.

---

USER AND SECURITY

* Built on ASP.NET Core Identity.
* User registration and login system.
* Each user can be associated with a sponsor (referrer).
* Users can add and update their IBAN information.

---

E-COMMERCE FEATURES

* Product listing
* Product detail pages
* Shopping cart system
* Order creation
* Order and order item management
* Stock and price tracking

---

REFERRAL AND BONUS SYSTEM

* Each user can directly invite a maximum of 2 users.
* The referral code becomes active after the user spends 5,000 TRY.
* Active referral count is updated automatically.
* For every 2 active direct referrals, a direct bonus is granted.
* Bonuses are not duplicated for the same referral pair.
* Chain bonuses are distributed up to 10 levels upward.
* Bonuses apply only within the user’s referral chain.
* The admin never receives any bonus or earnings.
* Total earning limit is 20,000 TRY.
* A one-time backup referral right is available.

---

WITHDRAWAL SYSTEM

* Users can submit withdrawal requests based on their earnings.
* Withdrawals are processed using the registered IBAN.
* The ToplamKazanc field is used as the real balance.
* The balance decreases after each withdrawal.
* Withdrawal request history can be viewed.

---

ADMIN PANEL

* Product create, update, and delete operations
* Category management
* Dynamic configuration of referral system rules
* Bonus reset and recalculation
* User and order monitoring
